mk-busy
===============

Busy spinner.

[Demo](http://mkant.ru/mink-js/mk-busy)

Inclusion
------------

Include style sheet into you page:
```HTML
<link href="path/to/plugin/mk-busy.min.css" rel="stylesheet">
```

Usage
-----------

```HTML
<div class="mk-busy-wheel"></div> 
```
